# CBOT For better Discord bots 

based on C 99 libary ( you can compile on C99, C11 C17) 

> [!WARNING]
> In the library you can build bots that comply with the Discord EULA and regulations. 
> The library is not responsible for incorrect implementation or non-compliance with Discord 
> regulations. You are responsible for your bot.

> [!IMPORTANT]
> To use the library, you must have the appropriate tools. Before launching, the bot must have an API compatible with our library and be compiled into an executable file (for Windows) or executable for Linux.
> Mac is not supported.

## Main Features
- Discord Connections
- Discord Bots 
- [GuiWidgets](readme-guiwidgets.md)
- [GameMaker](readme-GameMaker.md)
- [Own SocketConnector lib](readme-SocketConnector.md)
- C language support

# Examples

> [!NOTE]
> The basic configuration of your bot can be changed if necessary. If you don't know what you're doing, this is not recommended.
> If you not in UE country you can comment Location or = EXAMPLE ( In UE LOCATION for bridged ip REQUIRED you can comment this too if you not use briged ip )

```json
NAME_SERVER_ID = YOUR_DISCORD_ID_SERVER

LOCATION = YOUR_LOCATION_COUNTRY (IF YOU IN UE <- EU BRIDGED IP REQUIREMENT)

PERMISIONS_FOR_BOT = CHECK_IN_DISCORD_BASIC

DISCORD_REQ_BOT_KEY = YOUR_BOOT_KEY
```
